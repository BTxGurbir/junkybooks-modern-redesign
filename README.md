# junkybooks-modern-redesign
This project is a modern, futuristic redesign of the JunkyBooks.com website created as part of the Make It or Fake It event.
The goal was to improve the visual experience, modernize the UI, and enhance user interaction while keeping the original website structure.

Features:
-Dark premium futuristic theme
-Smooth animations and modern UI
-Book catalog with realistic layout
-Categories and browsing pages
-Responsive design for all devices
